## Installation

```sh
# Clone repository
git clone https://github.com/snuk0/loginster.git

# Move into directory
cd loginster

# Install dependecies
npm install
```

## Usage

```sh
# Set variables in .env.example

# Start in development mode
npm start

# Start in production mode
npm run start:prod
```
